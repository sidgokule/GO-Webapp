STEPS TO EXECUTE

1) Download and install Go & MySQL 
2) Extract the zip folder contents in the system to run the application
3) Open 'godb.sql' file in any MySQL interface and execute all the queries in given order to setup the  database
4) Open the main.go file in src/myGoWebapp folder and provide credentials for MySQL connection
5) Make sure to download all the packages imported in main.go file
6) Build GO file with following command in Command prompt/terminal: go build main.go
7) Run the executable with command: main.exe
8) Go to browser and enter URL: http://localhost:8080 to get to the landing page of application
7) Enter information and verify in the database
